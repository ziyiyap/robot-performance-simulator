# Robot Performance Simulator

A model that goes to the target location using unit vectors, noise included.

Features:

  ticks = time system
  
  CLI interface
  
  Performance included graphs, Eg. 'Battery vs Ticks', 'Temp vs Ticks', etc.
  
  Able to simulate multiple simulations.
  
  Able to switch between models.
  


Modules used:
numpy,matplotlib,pandas,json,pathlib
